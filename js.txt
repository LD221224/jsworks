* 자료형
        숫자, 문자(""), 불리언(true / false)
        , 배열([]), 참조 자료형

* 변수 
        값을 1개 저장, 변경 할 수 있음, var, let으로 선언
* 상수(constant variable)
        선언 후 변경 할 수 없음, const 예악어로 선언 & 대문자 사용
        사실, 진리, 변환 단위

* 연산자
        조건, 논리, 대입

* 배열 
        자료구조(대용량 자료)

* 제어문
        조건문 - if, switch
                if / else if / else
        반목문 - while, for
                while / break

* 내장 함수 
        write(), alert(), log(), push(), pop()
* 사용자 정의 함수  
        return이 있는 경우 / 없는 경우
        함수 정의
                function 함수이름(매개변수){
                        변수
                        명령문(if, for)
                }
        함수 호출
                함수이름(매개변수)

* 내장 객체
        Math, Date, Document, Window
* 객체(Class)
        생성자 함수, 참조 자료형, 사용자 정의 자료형
                function 클래스이름(){
                        변수
                        명령문
                        함수
                }
